To compile and run the TSA Screening Simulation there are 2 methods:

====Method 1:====

Navigate to the source folder and in a bash shell do:

./run.sh


If this does not work do:

chmod 755 run.sh
./run.sh


====Method 2:====

Navigate to the folder containing the java files in either a bash or cmd shell and do:

javac *.java
java Main
